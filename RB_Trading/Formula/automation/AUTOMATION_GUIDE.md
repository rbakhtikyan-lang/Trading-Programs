## 💾 БАЗА ДАННЫХ:

### Таблица `trades`:
```sql
id              Уникальный номер сделки
datetime        Время открытия
symbol          Монета (BTC/USDT)
direction       LONG или SHORT
entry_price     Цена входа
exit_price      Цена выхода
target_price    Целевая цена
stop_price      Стоп-лосс
position_size   Размер позиции (USDT)
leverage        Плечо
profit_usdt     Профит в USDT
profit_percent  Профит в %
status          OPEN / CLOSED / STOPPED
```

### Посмотреть в SQL:
```bash
sudo -u postgres psql -d trading_db
```

```sql
-- Все сделки за сегодня
SELECT * FROM trades WHERE DATE(datetime) = CURRENT_DATE;

-- Успешные сделки
SELECT * FROM trades WHERE profit_usdt > 0;

-- Статистика
SELECT 
    symbol,
    COUNT(*) as total,
    SUM(CASE WHEN profit_usdt > 0 THEN 1 ELSE 0 END) as wins,
    AVG(profit_usdt) as avg_profit
FROM trades
GROUP BY symbol;
```
