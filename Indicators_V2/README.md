# 📊 Trading Signal Bot (Rust)

## 🚀 Установка Rust (если не установлен)

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
source ~/.cargo/env
```

## ▶️ Сборка и запуск

```bash
cd trading_bot_rust
cargo build --release
./target/release/trading_bot
```

Или просто (сборка + запуск):
```bash
cargo run --release
```

## 🔧 Функции

- **Биржи**: Binance, Bybit, OKX
- **Индикаторы**: RSI, MA (20/50), MACD
- **Таймфреймы**: 15m, 30m, 1h, 4h
- **Сигналы**: LONG 🚀 / SHORT 📉 / НЕЙТРАЛЬНО ⚠️
- **Режимы**: разовый анализ или авто-обновление

## 📌 Логика сигналов

| Индикатор    | LONG                      | SHORT                     |
|--------------|---------------------------|---------------------------|
| RSI          | < 30 (перепродан)         | > 70 (перекуплен)         |
| MA Crossover | MA20 пересёк MA50 вверх   | MA20 пересёк MA50 вниз    |
| MA Тренд     | MA20 > MA50               | MA20 < MA50               |
| MACD         | Линия пересекла сигнал ↑  | Линия пересекла сигнал ↓  |
| MACD Hist    | Положительная             | Отрицательная             |
